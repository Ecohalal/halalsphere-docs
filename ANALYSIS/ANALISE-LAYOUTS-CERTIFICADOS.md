# Análise dos Layouts de Certificados FAMBRAS HALAL

**Data:** 2026-01-21
**Documentos Base:**
- `Exemplo-Certificado-unico.pdf` (AB Mauri Brasil - FM 7.7.2)
- `Exemplo-Aprovação-de-planta.pdf` (JBS - FM 7.7.1)

---

## Sumário

Este documento analisa os layouts oficiais dos certificados FAMBRAS HALAL para orientar a implementação da geração de PDFs no HalalSphere.

---

## Tipos de Certificado Identificados

| Tipo | Código | Uso | Páginas |
|------|--------|-----|---------|
| Certificado Único | FM 7.7.2 | Indústrias (alimentos, aditivos) | Múltiplas (1 + anexos) |
| Aprovação de Planta | FM 7.7.1 | Frigoríficos (abate) | Página única |
| Certificado de Embarque | FM 7.7.3 | Por lote/shipment | Página única |

## 3. Estrutura do Número do Certificado

### Padrão Identificado

```
[SIGLA].[UNIDADE].[AAMM].[SEQ].[VER?].BRA

Exemplos:
ABB.SOC.2402.5039.BRA
JBS.PCK.2307.1247.1.BRA
```

| Componente | Descrição | Exemplo |
|------------|-----------|---------|
| SIGLA | 3 letras da empresa | ABB, JBS |
| UNIDADE | Código da unidade/cidade | SOC (Sorocaba), PCK (Pontes/Lacerda) |
| AAMM | Ano (2 dígitos) + Mês (2 dígitos) | 2402 = Fev/2024 |
| SEQ | Número sequencial | 5039, 1247 |
| VER | Versão (opcional) | 1 |
| BRA | País | BRA (Brasil) |

### Comparação com HalalSphere

| Aspecto | FAMBRAS | HalalSphere Atual |
|---------|---------|-------------------|
| Formato | ABB.SOC.2402.5039.BRA | HS-YYYY-NNNNN |
| Inclui empresa | ✅ | ❌ |
| Inclui unidade | ✅ | ❌ |
| Inclui mês/ano | ✅ AA+MM | ✅ Só ano |
| Inclui versão | Opcional | ✅ Campo separado |

**Recomendação:** Manter formato HalalSphere internamente, mas considerar gerar número FAMBRAS para o PDF.

### Logos Requeridos

| Logo | Posição | Quando Usar |

## 8. Implementação Sugerida

### Backend (NestJS)

```typescript
// certificate-pdf.service.ts
@Injectable()
export class CertificatePdfService {
  async generateCertificatePdf(certificateId: string): Promise<Buffer> {
    const certificate = await this.certificateService.findOne(certificateId);
    const certification = await this.certificationService.findOne(certificate.certificationId);
    const company = await this.companyService.findOne(certification.companyId);
    const scope = await this.scopeService.findByCertificationId(certification.id);

    const templateType = this.determineTemplateType(certification);
    const template = this.loadTemplate(templateType);

    const data = {
      certificate,
      certification,
      company,
      scope,
      config: await this.getConfig(),
      qrCodeUrl: this.generateQrCode(certificate.certificationNumber),
    };

    return this.renderPdf(template, data);
  }

  private determineTemplateType(certification: Certification): string {
    // CV (Animal Slaughtering) = Aprovação de Planta
    if (certification.subcategory === 'CV') {
      return 'aprovacao_planta';
    }
    return 'certificado_unico';
  }
}
```
