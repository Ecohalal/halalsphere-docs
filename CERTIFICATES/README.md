# Certificados - Layouts e Referências

## Estrutura

- **layouts/** - PDFs de referência com layouts de certificados para análise
- Futuramente: análises, especificações de template, etc.

## Objetivo

Centralizar os modelos/layouts de certificados Halal para:
1. Analisar estrutura e campos obrigatórios
2. Definir templates para geração automática (IT 7.10)
3. Identificar variantes por mercado/país
